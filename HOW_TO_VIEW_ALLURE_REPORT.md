For better integration with Jenkins UI:

1. **Install Allure Plugin:**
   - Go to: **Manage Jenkins** â†’ **Manage Plugins** â†’ **Available**
   - Search for: `Allure Plugin` or `allure-jenkins-plugin`
   - Install and restart Jenkins

2. **Update Jenkinsfile** (after plugin installation):
   ```groovy
   stage('Publish Allure Report') {
   ```

   - After build, you'll see an **"Allure Report"** link in the build page
   - Click it to view the report directly in Jenkins UI
